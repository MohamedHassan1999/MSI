Door op bovenstaande document te drukken is het onderzoek in te zien dat ik voor Smart start heb gemaakt. In dit onderzoek heb ik de automatisering binnen Amazon onderzocht. Hiervoor heb ik onder andere een SWOT analyse gemaakt. Verder heb ik succesfactoren met meetniveau's opgesteld. Hiervan heb ik vooral geleerd dat bepaald werk verdwijnt door automatisering, maar tegelijkertijd op andere vakgebieden er juist werk bijkomt.

Binnen de minor heb ik diverse producten gemaakt. Deze worden via verschillende kanalen gecommuniceerd. In de PDCA-cyclus is aangegeven hoe dit wordt gedaan. In de volgende kolom staat welke communicatieboodschap op welke doelgroep gericht is. Van deze PDCA over communicatie heb ik geleerd dat via verschillende kanalen informatie overgebracht kan worden. Hoewel ik niet met elke methode bekend was voor deze minor (waaronder het bouwen van een website) ben ik wel blij dat ik deze kennis heb.

In deze kolom is te zien op welke doelgroep welke communicatiemethode uit de do-fase van de vorige kolom gericht is. Binnen deze communicatieanalyse is elke stakeholder binnen de minor meegenomen. Dit is gedaan door elke communicatieuiting uit de 4 stappen van de do-fase van de vorige kolom toe te lichten. Hiervan heb ik geleerd dat klasgenoten op hetzelfde niveau zitten als ik zelf en net zoals ik van hun kan leren hun ook van mij kunnen leren. Dit kan gedaan worden door kritisch naar elkaars deelproducten en eindproducten te kijken. De docenten kunnen vanuit hun expertise feedback geven waar ik uiteindelijk van kan leren. De opdrachtgever kan vanuit zijn organisatie probleemstelling aandragen, waar ik met mijn kennis en creativiteit oplossingen voor kan bedanken. Deze oplossingen kunnen onder andere via een presentatie of eindverslag gecommuniceerd worden.

Portfolio MSI Mohamed Hassan
