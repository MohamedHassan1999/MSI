Binnen deze minor heb ik kennis gekregen over big data en hoe deze in een bussines casus kan worden toegepast. Big data is een steeds groter wordend databestand die met meer velocity blijft toenemen. Vanuit een bussines prespectief kan deze informatie voor steeds meer toepassingen worden gebruikt. Voorbeelden hiervan zijn beter betere besluitvorming, productontwikkeling en geoptimaliseerde aanbiedingen. Door over een grote hoeveelheid data te beschikken kan een organisatie beter keuzes maken over welke strategie uitgevoerd moet worden. Deze informatie kan ook gebruikt worden om producten verder te ontwikkelen door de producten te verbinden met andere producten, zoals in een smart home systeem. Big data kan verder gebruikt worden om jouw klanten beter te leren kennen. Indien iemand vaak een bepaald product koopt kan een organisatie hiervan een persoonlijke aanbieding sturen naar deze klant. Klanten die lange tijd geleden een aankoop hebben gedaan kan een cadeaukaart worden gestuurd om deze terug te krijgen in de winkel.
